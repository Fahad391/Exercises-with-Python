This is a practice project with python.
used: Python OOP, Exception Handling, Numpy & Pandas, os

what Numpy Does here?
--> Analyze, calculate total cars, unique brands.

What Pandas does here?
--> Save data, Load data, add car into the collection and show cars collection.

what os does here?
-->  create File Path where data is stored

Created 2 Files --> Car_Analytics.py & main.py

in Car_Analytics.py:
1)created a Class Car_Tracker in it created a class variable file_path and given location where data is to be saved.
2) created a constructor under which os is utilized
3) created Methods _load_data, save_data, show_all_cars, add_car that used Pandas to load, store, display, add new cars
4) created Method get_total_counts, analyze that used NumPy to calculate and show Total Cars in the collection and How many Unique Brands

in main.py:
1) Imported Car_Tracker
2) Instantiated an object for the class Car_Tracker
3) created functions to manage methods of the class
4) created a while loop under which the main interface works.
